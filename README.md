# Django CRUD Application with Email & Image Upload

This is a Django-based web application that performs full CRUD (Create, Read, Update, Delete) operations with additional features like image upload and email notifications using SMTP.

---

## 🚀 Features

- Create, Read, Update, Delete (CRUD) functionality
- Image upload support
- Email sending functionality using SMTP (Office365)
- Bootstrap-based responsive UI
- Modal-based form handling (Add & Edit)

---

## 🛠️ Technologies Used

- Python
- Django
- HTML, CSS, Bootstrap
- SQLite (default database)
- SMTP (Office365 for email)
